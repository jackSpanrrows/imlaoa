## 条件表达式

```Tip```
```
适用于单行函数
```

## 定义:
```  
条件表达式是对于 if 语句的一种更为简短的句法规则. 例如: x = 1 if cond else 2.
```

## 优点:
    比 if 语句更加简短和方便.


## 缺点:
```
比 if 语句难于阅读. 如果表达式很长，难于定位条件.
```

## 结论:
```
适用于单行函数. 在其他情况下，推荐使用完整的if语句.
```
